# Used-Car-Market-Analysis

•	Analyzed 5,600+ used car records using Python, Pandas, and NumPy to uncover pricing patterns, regional variations, and customer preferences across Saudi Arabian markets.

•	Engineered data preprocessing pipeline with conditional imputation, categorical encoding, and log transformation to handle missing values, reducing price skewness and improving data quality significantly.

•	Created interactive Power BI dashboard visualizing regional price trends, brand value retention, and mileage impact, enabling data-driven strategic decisions for automotive market stakeholders effectively.

•	Discovered Toyota, Porsche, and Volkswagen maintained highest resale values, vehicles under 100,000 km commanded 15-20% price premiums, indicating strong buyer preferences and market dynamics.

•	Recommended vehicle condition maintenance strategies, low-mileage marketing advantages, and loyalty programs to strengthen buyer trust, brand reputation, and competitive positioning in automotive market.
